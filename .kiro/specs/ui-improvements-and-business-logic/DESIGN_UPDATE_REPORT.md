### 5. ✅ Адаптивность

**Добавлено:**
- Глобальные стили для адаптивности в `app.vue`
- Класс `.container-responsive` с breakpoints:
  - Mobile: 100%
  - SM (640px): max-width 640px
  - MD (768px): max-width 768px
  - LG (1024px): max-width 1024px
- Класс `.scrollbar-hide` для скрытия scrollbar
- Поддержка `safe-area-inset-bottom` для iOS
- Smooth scrolling

**Responsive элементы:**
- Grid layouts адаптируются на разных экранах
- Кнопки и карточки масштабируются корректно
- Текст остается читаемым на всех размерах
- Нижняя навигация адаптируется под safe area
